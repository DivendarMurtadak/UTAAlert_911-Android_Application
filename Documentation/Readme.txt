1. Install Eclipse IDE/Android Development Kit(if not already installed)
2. Click New -> select Android Application Project-> name the Project as "utaalert"
3. Set target between Android 2.2 (oldest API level 8) and Android 4.4.2(Latest API level 19). 
4. Set Java Compiler to 1.6
5. Unzip UtaAlert source code file-> copy all the files->replace it in the newly project created. 
6. Run using Emulator or using android mobile of version 2.2 or above.
